Документация {{ app_name }}

1) Введение

{{ app_name }} — веб‑приложение для ведения клиентской базы и статусов обслуживания у малого бизнеса.

2) Установка и запуск (Windows / Linux / macOS)

Вариант 1 — быстрый старт
pip install -r requirements.txt
python BizFlow
Вариант 2 — через виртуальное окружение
python -m venv .venv
source .venv/bin/activate   # Linux/macOS
.venv\Scripts\activate      # Windows PowerShell
pip install -r requirements.txt
python BizFlow
Далее откройте: http://127.0.0.1:5000

3) Основные функции

  • Регистрация и авторизация пользователей
  • Добавление, редактирование, удаление клиентов
  • Поиск и фильтр по статусу
  • Реляционная БД SQLite, данные сохраняются между запусками

4) Ручные тесты (3 сценария)

Тест 1
Регистрация и вход
Создайте аккаунт и войдите. Ожидаемо: открывается «Дашборд».
Тест 2
Некорректный ввод
Добавьте клиента с пустым ФИО. Ожидаемо: сообщение об ошибке, без падения.
Тест 3
Сохранение данных
Добавьте клиента, перезапустите сервер. Ожидаемо: клиент остался (SQLite).
